====================================================
Udemy Transcript Downloader - Chrome Extension
Udemy 課程逐字稿下載器
====================================================

功能特色:
- 批次下載 Udemy 課程逐字稿
- 按章節分檔輸出 TXT
- 可選擇包含時間戳記
- 支援下載全部合併為單一檔案
- 自動檢查登入狀態與課程存取權

====================================================
安裝步驟
====================================================

1. 生成圖示檔案:
   - 用瀏覽器開啟 icons/generate-icons.html
   - 右鍵點擊每個圖示 > 儲存圖片
   - 分別儲存為 icon16.png, icon48.png, icon128.png
   - 將這三個 PNG 檔案放到 icons 資料夾

2. 安裝擴充功能:
   - 開啟 Chrome 瀏覽器
   - 前往 chrome://extensions/
   - 開啟右上角的「開發人員模式」
   - 點擊「載入未封裝項目」
   - 選擇這個資料夾 (udemy-transcript-downloader)

3. 使用方式:
   - 前往任一 Udemy 課程頁面
   - 點擊瀏覽器工具列的擴充功能圖示
   - 勾選要下載的章節
   - 點擊「下載選取的章節」或「下載全部」

====================================================
檔案結構
====================================================

udemy-transcript-downloader/
├── manifest.json      # 擴充功能配置
├── popup.html         # 彈出視窗 HTML
├── popup.css          # 彈出視窗樣式
├── popup.js           # 彈出視窗邏輯
├── content.js         # 內容腳本 (擷取逐字稿)
├── background.js      # 背景服務工作者
├── README.txt         # 本說明檔案
└── icons/
    ├── generate-icons.html  # 圖示生成工具
    ├── icon16.png     # 16x16 圖示
    ├── icon48.png     # 48x48 圖示
    └── icon128.png    # 128x128 圖示

====================================================
注意事項
====================================================

1. 需登入 Udemy 並購買/擁有課程存取權
2. 課程必須有字幕/逐字稿功能
3. 下載過程中請勿關閉瀏覽器分頁
4. 僅供個人學習使用

====================================================
疑難排解
====================================================

Q: 找不到章節?
A: 請確保課程內容側邊欄已展開，可嘗試重新整理頁面

Q: 逐字稿顯示 [無逐字稿]?
A: 該講座可能沒有字幕，或是字幕尚未載入

Q: 下載失敗?
A: 請檢查 Chrome 的下載設定，確保允許下載多個檔案
